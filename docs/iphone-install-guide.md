# iPhone 安装使用指南

## 什么是"添加到主屏幕"？
"添加到主屏幕"是iPhone的一项功能，可以让网页像App一样出现在手机桌面上，点击即可打开，无需每次输入网址。

## 为什么推荐长辈使用？
- ✅ 像普通App一样使用
- ✅ 大字体、简洁界面
- ✅ 无需下载安装
- ✅ 不占手机存储空间
- ✅ 自动更新

## 安装步骤（图文说明）

### 第一步：在Safari中打开网站
1. 打开iPhone上的 **Safari浏览器**（蓝色指南针图标）
2. 在地址栏输入网站网址：`https://你的网址.com`
3. 点击"前往"打开网站

### 第二步：点击分享按钮
1. 点击屏幕底部的 **分享按钮** 📤
2. 分享按钮是一个方框带向上箭头的图标
3. 位于屏幕底部中间位置

### 第三步：找到"添加到主屏幕"
1. 在分享菜单中 **向下滑动**
2. 找到 **"添加到主屏幕"** 选项
3. 点击这个选项

### 第四步：确认添加
1. 可以修改显示名称（建议保持默认）
2. 点击右上角的 **"添加"** 按钮
3. App图标会出现在手机桌面上

### 第五步：完成安装
1. 返回手机桌面
2. 找到 **"妈妈灵修助手"** 图标
3. 点击图标即可打开使用

## 使用提示

### 如何打开应用？
- 点击手机桌面上的 **"妈妈灵修助手"** 图标
- 或者从Safari浏览器书签中打开

### 如何更新应用？
- 应用会自动更新，无需手动操作
- 每次打开时会检查更新

### 如果找不到"添加到主屏幕"选项？
1. 确保使用的是 **Safari浏览器**（不是微信、QQ等内置浏览器）
2. 确保网站已完全加载
3. 尝试刷新页面后重试

### 如果图标没有显示？
1. 检查网络连接
2. 尝试清除Safari缓存：
   - 打开"设置" → "Safari" → "清除历史记录与网站数据"
3. 重新访问网站并添加到主屏幕
